# Hack Assembly

This Project main goals is to introduce us to low-level programming through the Hack assembly language to interact directly to Hack hardware.

## Tasks

- [ ] Write Algebraic Program

  Our current hardware is only support an addition and a subtraction, we need a multiplication, use loop to make a multiplication possible.
- [ ] Write Interactive Program

  When a user press a key to keyboard (any key) the screen becomes black Vice Versa.
